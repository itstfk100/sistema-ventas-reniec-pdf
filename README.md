# Sistema de Ventas con RENIEC y PDF

## Descripción
Sistema web desarrollado con Spring Boot que permite registrar ventas,
consultar clientes por DNI usando una API externa (RENIEC) y generar
boletas en PDF automáticamente.

## Tecnologías
- Java 25
- Spring Boot 3
- MySQL
- Thymeleaf
- iText 7

## Funcionalidades
- Autenticación de usuarios
- Gestión de productos, usuarios y proveedores
- Registro de ventas con control de stock
- Generación de boletas en PDF

## Ejecución
1. Configurar la base de datos MySQL
2. Crear application.properties
3. Ejecutar el proyecto desde el IDE
